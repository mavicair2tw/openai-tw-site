AICRM 智能客服架構

此頁展示我們在 OpenClaw 上實作的高效率 AI Agent：具備 記憶蒸餾、技能凍結 與 模型降級策略，能在降低 40% Token 成本的前提下提供穩定客服體驗。

記憶蒸餾 Memory Distillation

  • 長對話 → <50 Token JSON 摘要
  • 蒐集 user_goals / preferences / constraints / patterns
  • 範例：{"user_goals":"Daily TW market report",...}

技能凍結 Skill Freezing

  • 重複任務自動變成技能模組
  • 內建：股市分析、Prompt 生成、API 呼叫、資料格式化
  • 技能先行 → 盡量不喚醒 LLM

模型降級策略 Model Downgrade

依照複雜度自動選擇模型：

if task in [chat, summarize, classify]: model = gpt-4o-mini
elif task in [code, reasoning]:       model = gpt-4o
else:                                   model = gpt-4.1

Agent Pipeline

  1. User Input → Distilled Memory
  2. Skill Router → 呼叫凍結技能
  3. 若需 LLM → Model Router 選擇最省成本模型
  4. 輸出後進行 Reflection，持續優化技能
