This page now reads from data.json instead of a hardcoded embedded snapshot. If you update that file with the included Python script, the dashboard can reload it in-browser.

  • Serve locally: cd taco-report && python3 -m http.server 8000
  • Open: http://127.0.0.1:8000
  • Use Reload data.json after regenerating the file.
